# Twitch 

Embedded Twitch code

----------------------------------------------------------------------------------------------------------------------------------------

- Import the zip files onto your local machine

```
git clone (https://github.com/aguin467/Twitch)
```

<p>&nbsp;
  

- Go into js folder

```
cd /Twitch/js
```

<p>&nbsp;
  

- Add channel name

```
var embed = new Twitch.Embed("twitch-embed", {
    width: 854,
    height: 480,
    channel: "Twitch Streamer Channel Name",
    layout: "video",
    autoplay: false
  });
```

<p>&nbsp;
  
- Enjoy! :thumbsup:
